# Basic Gulp Boilerplate

Boilerplate for very basic frontend needs. 

 - Gulp
 - Prettier
 - PostCSS (cssnano, autoprefixer, precss, postcss-preset-env)
 - Less
 - CSSNext

# Installation

```bash
git clone https://github.com/webMTD/basic-boilerplate.git
```
```bash
npm install
```
```bash
npm start
```
```bash
gulp watch
```
